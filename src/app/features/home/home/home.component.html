Acerca del proyecto

Esta aplicación es parte de un proyecto de portafolio que demuestra una arquitectura completa de tres capas:

  • 🐘 PostgreSQL 16 — base de datos relacional dockerizada
  • 🚀 NestJS — API REST con Sequelize y documentación Swagger
  • 🎨 Angular 18 — SPA con Material Design (este frontend)

Datos enfocados al contexto chileno: RUT, regiones y comunas oficiales.
