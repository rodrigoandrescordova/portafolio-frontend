Acerca del proyecto

Esta aplicación es parte de un proyecto de portafolio que demuestra una arquitectura completa de tres capas:

  • 🐘 PostgreSQL 16 — base de datos relacional dockerizada
  • 🚀 NestJS — API REST con Sequelize y documentación Swagger
  • 🎨 Angular 18 — SPA con Material Design (este frontend)
  • ¿Por Qué el formato — Uso para aprender Angular, ya que el resto lo veo día a día, modificando a diario por planteo de proyecto de personalización y ventas de medallas

Datos enfocados al contexto chileno: RUT, regiones y comunas oficiales, esto irá cambiando.
